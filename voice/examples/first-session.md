# 첫 세션 가이드

Comad Voice를 설치한 뒤, 처음 사용할 때 따라하세요.

## 1단계: 프로젝트 폴더에서 Claude Code 열기

터미널에서:
```bash
cd 내-프로젝트-폴더
claude
```

## 2단계: "검토해봐" 입력

```
검토해봐
```

Claude가 자동으로:
1. 프로젝트 전체를 살펴보고
2. 개선 가능한 영역을 카드로 보여줍니다

## 3단계: 카드 번호 선택

```
1번
```

선택한 카드의 개선 작업이 자동으로 시작됩니다.
실험 결과는 git commit으로 관리되며, 좋은 결과만 남깁니다.

## 4단계: 결과 확인

실험이 끝나면 Claude가 결과를 정리해서 보여줍니다.
만족스러우면:

```
다음 카드도 진행해줘
```

---

## 자주 쓰는 문장들

### 진단
- "검토해봐"
- "어디가 문제야?"
- "뭐 개선할 수 있어?"

### 실행
- "풀사이클" (대주제 자동 실행)
- "알아서 다 해줘"
- "1번 카드 진행해줘"

### 관리
- "메모리에 저장하고 새 세션"
- "지금까지 실험 결과 정리해줘"
- "다음 실험 미리 준비해줘"

### 병렬 작업 (Codex 있을 때)
- 자동 감지됨 — 직접 시킬 필요 없음
- Claude가 독립적인 작업을 알아서 Codex에 넘김

---

## 주의 사항

- **세션이 길어지면**: "메모리에 저장하고 새 세션" 입력
  (실험 5-7개가 적당한 한 세션 분량)
- **로컬 모델 쓸 때**: 테스트 대기 시간에 "다음 실험 준비해줘"
- **결과가 이상하면**: "이 결과를 분석해줘" (AI에게 해석 맡기기)
