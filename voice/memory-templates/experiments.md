# Experiment Log

각 실험의 핵심 결과만 기록. 새 세션에서 이전 실험 맥락을 빠르게 복원하기 위함.

## 작성 가이드

실험이 끝날 때마다 Claude가 아래 형식으로 자동 기록합니다.
수동으로 적어도 됩니다 — 핵심만 간결하게.

- **변경 사항**: 무엇을 바꿨는지 (코드 파일, 파라미터, 프롬프트 등)
- **가설**: 왜 이 변경이 효과가 있을 거라 생각했는지
- **결과**: 측정 가능한 수치 변화 (Before → After)
- **판단**: keep (유지) / discard (폐기) + 한 줄 이유
